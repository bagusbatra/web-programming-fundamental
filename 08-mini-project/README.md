# 🎯 Mini Project (Vanilla JavaScript App)
Vanilla JavaScript App adalah aplikasi web yang dibuat menggunakan JavaScript murni (tanpa framework atau library tambahan).

---

## 📌 Tujuan Pembelajaran

Setelah menyelesaikan project ini, kamu diharapkan:

- ✅ Mampu membuat aplikasi sederhana dari nol
- ✅ Menggabungkan HTML, CSS, dan JavaScript
- ✅ Mengelola data (array, object, localStorage)
- ✅ Mengambil data dari API (Fetch)
- ✅ Membuat UI interaktif & responsive
- ✅ Menulis kode yang rapi dan terstruktur

---

## 📖 Penjelasan Materi

### 🧠 1. Apa itu Mini Project?

Mini Project adalah:
> Aplikasi sederhana yang menggabungkan semua materi sebelumnya

Ini bukan latihan kecil lagi, tapi:
👉 Simulasi project nyata frontend

---

### 🔗 2. Materi yang Digabungkan

Project ini WAJIB menggunakan:

- HTML Semantic ✅
- CSS Flexbox & Responsive ✅
- Reusable Component ✅
- JavaScript DOM & Event ✅
- Array & Object (data) ✅
- localStorage ✅
- Fetch API ✅

💡 Intinya:
> Semua yang sudah dipelajari harus dipakai

---

#### Katalog Produk
Fitur:
- Tampilkan produk
- Tambah produk
- Hapus produk
- Filter produk
- Data dari API + lokal

---

👉 Bebas pilih, tapi harus memenuhi requirement

---

### 🏗️ 4. Struktur Project

Gunakan struktur sederhana:

    index.html
    style.css
    script.js
    assets/

💡 Intinya:
- Tidak perlu kompleks
- Fokus ke logic & UI

---

### ⚙️ 5. Alur Aplikasi (WAJIB PAHAM)

Setiap app harus punya alur:

1. Ambil data (array / API)
2. Tampilkan ke UI
3. User interaksi
4. Update data
5. Render ulang

👉 Ini disebut:
**Data → UI → Event → Update → Render**

---

### 💻 6. Contoh Pola Kode

#### Data
```js
    let data = [];
```

---

#### Render
```js
    function render() {
        // tampilkan data ke HTML
    }
```

---

#### Event
```js
    button.addEventListener("click", function() {
        // update data
        render();
    });
```

---

💡 Intinya:
- Pisahkan logic & tampilan
- Jangan campur semua di 1 tempat

---

### 🎨 7. UI & UX

Gunakan:

- Card layout
- Button reusable
- Spacing rapi
- Responsive

💡 Ingat:
> UI bagus = lebih mudah dipahami user

---

### 🌐 8. Integrasi API

Gunakan:
```
    fetch()
```

Untuk:
- Ambil data tambahan
- Kombinasikan dengan data lokal

---

### 💾 9. Simpan Data

Gunakan:
```
    localStorage
```

Tujuan:
- Data tidak hilang saat refresh
