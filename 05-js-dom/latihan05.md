# 📝 Latihan

## 🎯 Tujuan

- Menghubungkan UI (HTML + CSS) dengan JavaScript
- Memahami manipulasi DOM secara nyata
- Menggunakan event untuk interaksi user
- Mulai berpikir logic sederhana (bukan sekadar klik)
- Mengembangkan project sebelumnya menjadi interaktif

---

## 📌 Instruksi

Gunakan hasil dari **Latihan 04 (Reusable UI System)**, lalu tambahkan **interaksi menggunakan JavaScript**.

---

### 1. Interaksi Button (Wajib)

Tambahkan minimal 2 fitur:

#### a. Ubah Teks
- Saat tombol diklik:
  - Judul / deskripsi berubah

#### b. Toggle Element
- Saat tombol diklik:
  - Section bisa muncul / hilang

---

### 2. Interaksi Card

Tambahkan interaksi pada card:

- Saat card diklik:
  - Card berubah warna
  - ATAU
  - Card menampilkan pesan tambahan

👉 Tidak boleh hanya 1 card (harus semua card aktif)

---

### 3. Input User (REAL CASE)

Buat fitur:

- User input nama
- Klik tombol → tampilkan di halaman

Contoh:
👉 "Halo, Bagus!"

---

## ⭐ Challenge

🔥 Level Up (mulai masuk logic):

### 1. Toggle Active Card

- Saat card diklik:
  - Tambahkan class `active`
- Jika diklik lagi:
  - Kembalikan seperti semula

👉 Hint:
Gunakan `classList.toggle()`

---

### 2. Validasi Input (PENTING)

Jika input kosong:
- Tampilkan pesan:
  👉 "Nama tidak boleh kosong!"

Jika diisi:
- Tampilkan nama

---

### 3. Interaksi Lebih “Real”

Tambahkan minimal 1 fitur baru:

Contoh:
- Klik tombol → tambah card
- Klik card → hapus card
- Toggle semua card

---

## 💡 Tips

- Gunakan `console.log()` untuk debug
- Pastikan selector benar
- Pisahkan logika per fitur
- Jangan campur semua di 1 function
